Central Limit Theorem. In this demo, each value is drawn independently from the same population. Individual values can be lopsided, flat, or split into groups. As each sample average uses more values, the distribution of those averages becomes more mound-shaped around the true average. The spread of the sample averages is about population SD / sqrt(sample size). For small sample sizes or very lopsided populations, the bars can still look bumpy before the approximation becomes clear.
